HTML

HTML (Hyper Text Markup Language) temelj je svake stranice. Jednostavan opisni markup jezik koji prezentira raspored sadržaja na stranici. Sastoji se od "naredbi",odnosno tagova. Moguće ga je pisat u bilo kojem tekstualnom editoru, uključujući i Notepad.
HTML 5 donosi i neke nove elemente poput: crtaće površine ‹canvas›, umetanje videa ‹video›. Donosi nove tipove unosa u obrascima, a neki od njih su: search, number, range, color, email, url. HTML 5 također pruža opcije za predupisani tekst, (placeholder text), te automatsko postavljanje fokusa, (autofocus). Dostupni su nam i novi semantički elemnti: ‹header›, ‹section›, ‹nav›, ‹article›, ‹aside›, ‹hgroup›, ‹footer›, ‹time› i ‹mark›.

CSS3

CSS (Cascade Style Sheet) je opisni jezik, koji opisuje dokument napisan markup jezikom. Osim za izgled, CSS definira i raspored elemenata na stranici. Konstantno se razvija s obzirom na potrebe samoga web dizajna. Podržan je od strane svih popularnih pretraživača za web.

jQuery

jQuery je JavaScript biblioteka podržana od strane svih popularnih pretraživača. Osmišljen kako bi se pojednostavile klijentske scripte na stranici HTML-a. Omogućuje stvaranje moćnih, dinamičkih web stranica i web aplikacija.

ASP.NET

ASP.NET je razvojna okolina koja omogućuje programerima razvijanje web aplikacija. Objektno je orijentiran što ga približava objektno orijentiranim jezicim poput: C#, VB i drugih. Postojeće ASP aplikacije moguće je pobošljavati ASP.NET elementima.

SEO OPTIMIZACIJA

SEO (Search engine optimization), odnosno optimizacija za tražilice, dugačak je proces kako bi bili što bolje pozicionirani sa određenim ključnim riječima. U današnje vrijeme, pored tolikoga broja web stranica, nužna je kvalitetna optimizacija ukoliko želite biti viđeni na internetu. Za početak potrebno je poštivat neke osnovne standarde koji utječu na tražilice. Neke od osnovnih stvari su:
‹title› i ‹meta description› tagove za svaki url koji postoji na stranici. Ukoliko tagovi nisu ispunjeni, tražilica automatski stavlja prvi naslov u ‹title› tag, a za opis stranice prvih 160 znakova sadržaja.
‹h› tagovi moraju omotavati naslove stranice, tekstova, članaka. Alt i title tagovi kod slika koji će je opisivati. Korištenje favicon.ico (ikona od 16x16 piksela) koja je smještena u root stranice. HTML kod bi trebao biti ispravan, svi linkovi prekontrolirani da su u funkciji. Naslovna stranica nebi smjela prelaziti preko 180Kb. Ovo su samo neke osnovne stvari za kvalitetan početak SEO optimizacije.
Kasnije bi se trebali pozabaviti sitemap-om, robots.txt datotekom, analiziranjem ključnih riječi, google webmaster alatima te naravno marketingom, odnosno promoviranjem svoje web stranice.

AJAX

Asynchronous JavaScript and XML (AJAX), u zadnjih nekoliko godina sve više popularnija tehnologija za kreiranje boljih i prije svega bržih web aplikacija. Također, jedna od prednosti upotrebe AJAX-a je izrada jednog dijela stranice, sadržaja, u toj tehnologiji. Tada je omogućeno ovježavanje samo tog djela stranice, bez potrebe ponovnog učitavanje cijele web lokacije. Pri izvršavanju naredbi koristi asinkroni transfer podataka između browser-a i servera.

WordPress

WordPress je jedan od najpopularnijih programskih kodova otvorenog sadržaja (open source), za izradu i održavanje web lokacija, te moćan sustav za upravljanje sadržajem. Izuzetno je prilagodljiv svakom korisniku. Jednostavan je za podignuti ga, te još lakše njime upravljati.
Postoje dvije varijante WordPress-a.
1. Besplatna verzija na WordPress.org, osmišljena prvenstveno za vođenje blogova.
2. Verzija WordPress-a koju morate skinuti sa lokacije WordPress.org i instalirati na server. Ova verzija osmišljena je za izradu interaktivnih web stranica.
